## 這個部落格是怎麼產生的？

其實在我國高中的時候，身邊的電神們很多都有自己的 Blog 或者是 Personal Web，所以我認為我在這個時間點寫已經有億點點晚了。

這幾天剛好有郁秀杯，而我負責的是全端的部份。雖然其實我根本對於 web 沒有什麼概念，不過前幾天一邊查文件、一邊慢慢試，發現要把一個網站整理出來好像也沒有到很困難。在這之前也因為沈迷了**演算法**許久根本沒有心力寫自己的 web 不過當然你也可以當作是藉口XD


總之這個網站比較像是把自己想要的空間慢慢搭起來。從排版、文章、導覽到之後想放的內容，都會慢慢調成比較像我自己的樣子。

總之，感謝你看到這邊。如果你對我很有興趣，在我的熱情消耗之前也許你可以偶爾近來逛逛，應該時不時會出現幾個 Blog 或是隨筆希望你可以來看看。

## 關於我 Blog 的期望
其實在眾多的部落格中，我最喜歡的部落格應該就是 [wiwi.blog](https://wiwi.blog/)，官老師一直對於網路有很強烈的看法，包括脫離 Youtube 跑到自己的串流，甚至連 GitHub 都可以捨棄。對我來說這是一件很難以想像的事情，也希望之後在這個部落格也可以不只有一堆 code 甚至有對於我對音樂/遊戲/人生的見解。不過當然，目前我只是一個來自新北的普通高中生，如果對於我的 Blog 有任何的問題都歡迎透過任何管道和我說，我都會虛心指教。
